КРАТКОЕ ОПИСАНИЕ:
Проект "Сайт для танцевальной студии" представляет собой веб-сайт, разработанный в соответствии с заданием, 
включая необходимые компоненты и адаптивную верстку. Сайт предоставляет информацию о студии, 
расписании занятий, акциях и контактной информации.

ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА:

* Главная страница: представлена информация о проходящих акциях,танцевальных направлениях, расположении и стоимости.

* Расписание: предоставляет информацию о доступных занятиях и их времени.

* Pole Dance: прдоставлена информация о конкретном танцевальном направлении, а также расписание для этого направления. 


ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ:

* HTML и CSS: Верстка и стилизация веб-страниц с помощью HTML и CSS.

* Figma: Макет сайта был разработан в Figma, и все дизайн-решения были взяты из макета, включая цветовую палитру и шрифты.

* Адаптивная верстка: Сайт разработан с учетом адаптивности для мобильных устройств и планшетов. 
Использованы контрольные точки для обеспечения корректного отображения на разных экранах.

ОСОБЕННОСТИ ПРОЕКТА:

* Проект максимально приближен к макету в Figma, включая использование цветов, скругление границ, фон и шрифты.

* Все необходимые изображения были скачаны из макета в Figma и использованы на сайте.
